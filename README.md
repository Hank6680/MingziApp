# MingziApp

MingziApp 是 Aeron 的 React + Vite + TypeScript 项目骨架，用来快速验证前端交互与组件。

## 技术栈
- React 19 + TypeScript
- Vite 7 开发服务器与构建工具
- ESLint + TypeScript ESLint 规则集

## 快速开始
1. 安装依赖：`npm install`
2. 启动开发服务器：`npm run dev`
3. 在浏览器打开日志中提示的本地地址（默认 http://localhost:5173）即可预览。

## 构建 & 其他脚本
- `npm run build`：生成生产构建产物
- `npm run preview`：预览生产构建
- `npm run lint`：运行 ESLint 检查

当前模板不需要环境变量，如需新增配置，请在项目根目录创建 `.env` 并在本文档补充说明。
